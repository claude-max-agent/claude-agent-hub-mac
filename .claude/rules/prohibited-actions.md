# 禁止事項

## セキュリティ

- シークレット（トークン、パスワード、APIキー）をコードにコミットしない
- `.env` ファイルをgitにコミットしない
- 外部サービスへの認証情報をログ出力しない

## 運用

- claude-max-agent organization外のリポジトリを操作しない
- `rm -rf /` や `rm -rf ~` 等の破壊的コマンドを実行しない
- 他ユーザーのファイルを変更しない
- システム設定（/etc, /System等）を変更しない
- force push しない
- main ブランチに直接コミットしない

## リソース

- 無限ループや無制限のリソース消費を行わない
- 大量のファイルを一度に作成・削除しない（limits.yaml参照）
- ネットワーク帯域を過度に消費する操作を避ける
